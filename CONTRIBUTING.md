# Contributing

Dit is een persoonlijk project en ik neem geen externe bijdragen aan. Issues, pull requests en discussions zijn uitgeschakeld op de repo.

De code staat publiek voor transparantie en zodat anderen ervan kunnen leren — niet als open invitatie voor samenwerking. Voel je vrij om de patronen over te nemen voor je eigen projecten (de [MIT-licentie](LICENSE) staat dat expliciet toe).

Heb je iets gevonden dat echt aandacht verdient — een security-issue, een bug die mij in de problemen kan brengen — kijk dan in [`SECURITY.md`](SECURITY.md) hoe je dat verantwoord meldt.

Bedankt voor het rondkijken.
