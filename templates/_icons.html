{# =========================================================================== Sluiskade · icon library PROJECT CONVENTION: this project NEVER uses emoji characters in UI. Emojis render inconsistently across operating systems, look out of place on a custom-designed page, and are not stylable. Use these SVG icons instead. Pick from the list below or add a new one to this file · do not inline emoji anywhere in templates, JS, or copy. All icons use the Lucide line-icon style: 24x24 viewBox, 1.5px stroke, rounded line caps, currentColor so they inherit text colour. Size them with CSS by setting width/height (or font-size if you pass `class`).
